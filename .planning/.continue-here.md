## Key Decisions Made

1. **Decoupled architecture**: Express serves JSON API, React is a separate SPA
2. **URL-based user selection**: `/tracker?user=ID` instead of sessions, making profiles shareable
3. **SVG map extraction**: 176 country paths extracted from EJS to `mapData.json` and rendered via React component
4. **No Tailwind**: Pure vanilla CSS with CSS custom properties for the design system
5. **Lucide React icons**: No emojis in UI, all proper SVG icons
