# Project: Family Travel Tracker Refactoring

**What This Is:** A full-stack web application designed to track and visualize visited countries for various family members. This project refactors the existing application with a modern responsive UI, a clean database layout, and user session management so family members can see all members added when logged in.

## Stack
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL (`pg` database client)
- **Frontend:** HTML, EJS, Vanilla CSS (harmonious dark/light theme, modern typography, glassmorphism, responsive, interactive map)
- **Security:** `dotenv` for securing credentials

## Core Value
To provide family members with a beautiful, fast, and intuitive portal to log and share their travel histories.
